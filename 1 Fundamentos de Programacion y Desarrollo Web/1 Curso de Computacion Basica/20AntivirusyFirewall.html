El antivirus y el firewall son programas que mantienen seguro tu computador, controlando el ingreso de datos maliciosos. Conocerlos es fundamental para tener una mejor seguridad informática. En la barra de búsqueda de Windows ingresa “Antivirus” y escoge la opción “Protección contra virus y amenazas”. Vista de configuración de amenzas El antivirus te provee de opciones para revisar tu computador de posibles amenazas, el tiempo que dure este análisis depende de tu hardware y la cantidad de archivos. Tipos de examenes para el computador En la barra de búsqueda de Windows ingresa “Firewall” y escoge la opción “Windows Defender Firewall con seguridad avanzada”. Ventana de configuración de Firewall Este programa se encarga de revisar y mostrar todo el tráfico de quienes se conectan desde tu computador a internet, y viceversa. Los programas necesitan permisos para conectarse a internet, por eso al momento de instalarlos te piden un acceso al Firewall del sistema. De esta manera tienes el control de los programas que acceden a tu computador o se conectan a internet.
