La interfaz de Windows está conformada por varios elementos gráficos para el usuario, como el fondo de escritorio y la barra de tareas. Elementos de la interfaz gráfica de Windows Imagen personalizable que ocupa toda la pantalla cuando se encuentra en el escritorio de Windows. Descarga el fondo de escritorio de “Nunca pares de aprender”. Conjunto de elementos en el que Windows muestra información relevante para tu día a día. Elementos de la barra de tareas lado izquierdo Elementos de la barra de tareas lado derecho Para configurar los dispositivos Bluetooth en Windows 10 realiza lo siguiente: Selecciona el botón Inicio, luego selecciona Configuración > Dispositivos > Bluetooth y otros dispositivos. Te encontrarás con una ventana similar a esta: Pantalla de Bluetooth y otros dispositivos Se debe tener en cuenta que algunas computadoras que no tienen el hardware para conectarse vía Bluetooth, esa opción estará desactivada. Para estos casos es necesario comprar un adaptador Bluetooth. El mouse tiene dos tipos de clic: el principal, que sirve para manejar la interfaz gráfica; y el contextual, que sirve para abrir ventanas de opciones o de contexto. Partes de un mouse Dentro de la configuración de dispositivos, en la opción de Mouse, te encontrarás con una ventana similar a esta: Pantalla de configuración del mouse En “Seleccionar el botón primario” se configura el clic principal, que por defecto es el clic izquierdo. Para personas zurdas, configurar esto es muy importante. Ícono de Idioma de la barra de tareas Al dar clic al apartado de idioma en la barra de tareas y luego “Preferencias de idioma”, te aparecerá la siguiente ventana en el que podrás descargar e instalar diferentes lenguajes según tu preferencia.
