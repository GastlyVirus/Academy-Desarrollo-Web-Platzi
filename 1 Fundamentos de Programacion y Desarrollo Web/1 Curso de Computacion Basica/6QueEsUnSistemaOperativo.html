Los sistemas operativos son el software de sistema que se encarga de comunicarse entre tu hardware y las aplicaciones que usas. Los sistemas operativos utilizados en computadores son Windows, MacOS y Linux. MacOS es el sistema operativo de Apple para sus computadoras de escritorio y portátiles. Mac OS son las siglas de “Macintosh Operating System” o lo que es lo mismo “Sistema Operativo Macintosh”. Este sistema operativo es exclusivo para los dispositivos producidos por Apple. MacOS Monterey Windows es el sistema operativo de propósito general más utilizado a nivel mundial, es un sistema operativo cerrado y se encuentra en la gran mayoría de computadoras para consumidores (HP, Lenovo, Asus, etc.), además utiliza un núcleo propietario perteneciente a Microsoft. Windows 10 Windows 11 Linux es el sistema operativo más empleado en servidores y de software libre. Su creador Linus Torvalds aún desarrolla su núcleo, destacado por su alto rendimiento y alta seguridad. Este sistema operativo tiene una licencia del tipo GNU-GPL que no solo permite redistribuir, sino también garantiza que las personas que redistribuyen el código deban aportar a la licencia, entre otras cosas. Generalmente, las computadoras vienen con una licencia Windows oficial ya instalada, esta es más que suficiente. La seguridad del sistema operativo depende de qué tan actualizado esté. Todos los sistemas operativos poseen errores de seguridad, por lo que su fabricante (Windows, Apple, etc.) resuelve con pequeñas actualizaciones de seguridad que arreglan las vulnerabilidades que pudieran existir. Por lo que, si tienes un software pirata, estas actualizaciones no llegarán y estarás vulnerable a ataques de seguridad.
