El software es el término informático para un programa o conjunto de programas que manejan operaciones dentro de la computadora. Ejemplos de software de aplicación son: Google Chrome, Platzi, Word, Excel, Minecraft, entre otros. Un programa que busca obtener información de nuestra computadora, provocando que tus datos sean dañados, encriptados o borrados. Por ejemplo, programas que cierren ventanas, que cambien tus fotos por las de alguien más, o que guarden la información que ingreses por teclado al momento de iniciar sesión en una cuenta bancaria y así obtener tu contraseña. El software libre (Open Source) es código público, que cualquier persona puede acceder, usar, contribuir o modificar respetando los términos de licencia de la empresa o del desarrollador. Por ejemplo: Ubuntu. El software propietario es código privado, es decir, solamente aquellos que tienen permiso pueden acceder, usar, contribuir o modificar este software. Por ejemplo: Windows, Platzi. Los software de sistema o controladores (drivers) son programas que conectan el hardware de tu computadora al sistema operativo para que los entienda y conceda permisos para usarlos. El software de programación es el código que se escribe en cualquier lenguaje de programación para comunicar todo el software y mandar las instrucciones que la computadora entiende mediante el sistema binario. Descargar software de los sitios oficiales de la empresa o de fuentes confiables que sigan un protocolo de seguridad. Tener cuidado con el hardware que ingresamos a nuestro computador, como una USB, pueden contener software malicioso.
