Al usar software pirata estás vulnerable a cualquier tipo de ataque, por eso es recomendable usar software oficial pagando por el mantenimiento y seguridad que ofrecen los propietarios del servicio. Aunque no es recomendable y legal, existen programadores que ofrecen un software pirata seguro, en esos casos e informándote bien acerca de lo que vas a instalar, puedes instalar el software hasta que tengas la posibilidad de pagar. Todo programa tiene vulnerabilidades de seguridad, por eso Windows mantiene su software con pequeñas actualizaciones que resuelven estas inseguridades. Para actualizar Windows, en la barra de búsqueda ingresa “Buscar actualizaciones”. Se abrirá “Windows Update” que te muestra las actualizaciones disponibles de tu computador. Ventana de actualización Windows Update Una vez descargadas e instaladas las actualizaciones, es necesario reiniciar el computador para aplicarlas. Puedes hacerlo al momento con la opción “Reiniciar ahora” o después programando el reinicio. Ventana para programar una hora de reinicio
